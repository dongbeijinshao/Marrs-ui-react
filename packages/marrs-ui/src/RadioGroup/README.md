### CheckboxGroup Props

| 参数         | 说明                     | 类型                 | 默认值  |
| ------------ | ------------------------ | -------------------- | ------- |
| className    | 额外 class 名称          | --                   |
| classes      | 替换 class               | -{ root: my-class }- |
| icon         | 默认值                   | ``                   |
| row          | 是否横向排列方向         | _boolean_            | `false` |
| checkedColor | 所有单选框的选中状态颜色 | _theme_              | --      |
| disabled     | 是否禁用所有单选框       | _boolean_            | -false- |
