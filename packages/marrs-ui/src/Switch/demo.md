# Switch 开关

### 介绍

用于在打开和关闭状态之间进行切换。

### 引入

```js
import { Switch } from '@wemo-ui/marrs';
```

## 代码演示

### 基础样式

```html
<Switch checked />
```
### 禁用状态

```html
<Switch disabled checked />
```
### 加载状态

```html
<Switch loading checked />
```

### 自定义颜色值

```html
<Switch className="demo-margin-r-10" checkedColor="#2A6AE9" checked />
<Switch className="demo-margin-r-10" checkedColor="#07C160" checked />
```
